# BraineBase - Knowledge Management System

## Project Description
BraineBase is a powerful and intuitive Knowledge Management System that helps individuals and organizations manage their knowledge efficiently. This system allows users to store, organize, and retrieve information quickly, facilitating better decision-making and collaboration.

## Features
- **User Authentication:** Secure login and registration for users.
- **Knowledge Base Creation:** Create, edit, and manage knowledge articles.
- **Search Functionality:** Powerful search to quickly find articles and resources.
- **Categorization:** Organize articles into categories for easier navigation.
- **Collaboration Tools:** Share articles and collaborate with other users.
- **Version Control:** Keep track of changes made to articles.

## Tech Stack
- **Frontend:** React.js
- **Backend:** LARAVEL ON https://github.com/OuafikMohammed/BrainBase-Backend
- **Authentication:** JWT (JSON Web Tokens)
- **Styling:** CSS, Bootstrap

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/OuafikMohammed/BraineBase---Knowledge-Management-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd BraineBase---Knowledge-Management-System
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Set up the database and environment variables as described in the `env.example` file.
5. Start the server:
   ```bash
   npm start
   ```
